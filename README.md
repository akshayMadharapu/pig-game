This is a simple Pig Game that I built using HTML, CSS, and JavaScript. I created this project to practice JavaScript concepts like DOM manipulation, event handling, variables, arrays, and game logic.

Features
Two-player game
Random dice roll
Current score updates after every roll
Rolling a 1 resets the current score and switches the turn
Hold button to save the current score
First player to reach 10 points wins
New Game button to restart the game
Technologies Used
HTML
CSS
JavaScript
How to Play
Click Roll Dice to roll the dice.
If the dice value is 2 to 6, it gets added to your current score.
If you roll a 1, your current score becomes 0 and the turn goes to the other player.
Click Hold to add your current score to your total score.
The first player to reach 10 points wins the game.
